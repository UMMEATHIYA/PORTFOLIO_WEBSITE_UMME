AI/ML and System Design Projects

Privacy-Preserving LLM Platform
This project focused on addressing privacy concerns related to LLMs in enterprise environments. I worked on designing a system that leveraged homomorphic encryption (FHE) to secure the data during inference, without compromising model performance. We were able to fine-tune a 70B parameter model using QLoRA and reduced costs by 8x. A major accomplishment was integrating the model into AWS SageMaker for secure and scalable deployment, serving 200+ users in a production environment.
Enterprise GenAI Platform
This project involved the orchestration of multiple LLMs (including GPT-4 and Llama 2) to serve dynamic content generation in real-time. The focus was on creating a flexible and cost-efficient system that could scale. One key feature I developed was dynamic model routing, which resulted in a 40% cost reduction in the overall inference process while maintaining a 99.9% service-level agreement (SLA). This platform significantly impacted the company's bottom line, reducing operational costs by over $2.3 million annually.
National-Scale Computer Vision Deployment
I led the development of a computer vision system deployed across 8 manufacturing hubs, processing over 15 million inferences daily. The system utilized YOLOv9 for real-time object detection in quality control processes. Additionally, I integrated a federated learning pipeline that adhered to strict data compliance regulations (GDPR, CCPA, etc.) across multiple regions. The deployment also incorporated Kubernetes for container orchestration and Istio for service mesh, ensuring 99.97% uptime, even during peak usage.
Real-time Fraud Detection System
This project involved designing and deploying a fraud detection system that analyzed over 5 million transactions per month. The system was built using YOLOv8 for anomaly detection, combined with Kafka for real-time data streaming. We implemented a scalable architecture that integrated multiple data sources, allowing the system to identify fraudulent transactions with high precision. As a result, we prevented over $2 million in annual fraud losses by accurately flagging high-risk transactions within milliseconds of being processed.
Federated Learning for IoT Devices
I developed a federated learning framework for IoT devices that ensured privacy preservation while training on decentralized data. The project aimed to build a predictive maintenance model across several edge devices without compromising data privacy. We overcame challenges such as data heterogeneity and model aggregation by developing a custom aggregation algorithm. This solution allowed for more accurate predictions of equipment failure, with a reduction in downtime by 25% and improved maintenance efficiency across the organization.
